# Excel & Power BI Business Analytics

## Описание проекта

Провести комплексный анализ данных с использованием Excel, VBA и Power BI с целью обработки информации, расчёта бизнес-метрик, автоматизации аналитических процессов и визуализации ключевых показателей для принятия решений.

В рамках проекта выполнены следующие задачи:

- обработка и структурирование данных в Excel  
- расчёт ключевых бизнес-метрик (выручка, ROI, показатели эффективности)  
- работа со сводными таблицами и аналитическими отчётами  
- автоматизация расчётов и процессов с помощью VBA (макросы)  
- подготовка данных для дальнейшего анализа  
- построение дашбордов и визуализаций в Power BI  
- анализ данных и формирование выводов  

## 📂 Исходные данные

В работе используются данные по заявкам, содержащие:

- `Внешний номер` — идентификатор заявки  
- `Банкомат` — ID банкомата  
- `Регион` — регион  
- `РОО` — подразделение  
- `Получатель` — получатель заявки  
- `Тип сбоя` — тип проблемы  
- `Дата отправки` — дата начала работ  
- `Дата окончания работ` — дата завершения работ  
- `В расчете доступности` — учитывается ли заявка в доступности  
- `Просрочено` — признак просрочки  
- `Повторно` — повторная заявка  
- `Платно` — платная заявка  
- `Отменено` — статус отмены

## 🛠 Задача 1. Обработка заявок 

В рамках задания необходимо:

1. Определить заявки, которые присутствуют на листе `WO`, и указать признак:
   - присутствует 
   - отсутствует 

2. Для заявок, присутствующих на листе `WO`, дополнительно подтянуть:
   - ID банкомата  
   - дату начала работ  
   - дату завершения работ  

Решение реализовано с использованием функции `ВПР (VLOOKUP)`.

- с признаком наличия заявки

## 🛠 Задача 2. Анализ заявок по РОО

В рамках задания выполнен анализ заявок по подразделениям (РОО):

1. Определён перечень уникальных РОО (столбец `C`), присутствующих на листе `WO`  

2. Рассчитано количество заявок по каждому РОО  

3. Определена средняя длительность заявок по всем РОО  
   *(разница между датой окончания и датой отправки)*  

4. Рассчитан общий фонд времени заявок по каждому РОО
