# Excel & Power BI Business Analytics T1 (Холдинг)

## Описание проекта

Провести комплексный анализ данных с использованием Excel, VBA и Power BI с целью обработки информации, расчёта бизнес-метрик, автоматизации аналитических процессов и визуализации ключевых показателей для принятия решений.

В рамках проекта выполнены следующие задачи:

- обработка и структурирование данных в Excel  
- расчёт ключевых бизнес-метрик (выручка, ROI, показатели эффективности)  
- работа со сводными таблицами и аналитическими отчётами  
- автоматизация расчётов и процессов с помощью VBA (макросы)  
- подготовка данных для дальнейшего анализа  
- построение дашбордов и визуализаций в Power BI  
- анализ данных и формирование выводов

## 🛠 Задача 5. Выявление аномальных отклонений

В рамках задания выполнен анализ количества ошибок по типам во времени с целью выявления аномальных отклонений.

Для каждого типа ошибки:
- определено среднее значение  
- проанализированы ежедневные отклонения  
- выявлены выбросы относительно нормального уровня  

Аномалии выделены с помощью условного форматирования.

Для выявления отклонений использовано правило условного форматирования на основе статистических показателей:

- верхняя граница:
  
  `значение > среднее + 1,5 × стандартное отклонение`

- нижняя граница:
  
  `значение < среднее − 1,5 × стандартное отклонение`

   **жёлтый цвет** — аномально высокие значения  
   **голубой цвет** — аномально низкие значения  

Коэффициент `1,5` выбран вместо `2`, так как при более строгом пороге выбросы не выявлялись, а текущее значение позволяет зафиксировать значимые отклонения.
